Google Sheets

Conecte via Google Apps Script (intermediário seguro). Todos os produtos e estoque são salvos em tempo real na sua planilha.

Não conectado
Como configurar (5 passos):
1. Acesse script.google.com → Novo Projeto
2. Cole o conteúdo do arquivo apps-script.js entregue junto
3. Salve (Ctrl+S) e clique em Implantar → Nova Implantação
4. Tipo: Aplicativo da Web · Executar como: Eu · Acesso: Qualquer pessoa
5. Copie a URL gerada e cole no campo acima
☁️

Cloudinary

Use o Cloudinary para hospedar todas as imagens dos produtos. Upload automático, otimização e CDN global inclusos.

Não conectado
Crie em: Cloudinary → Settings → Upload → Upload Presets
Como configurar:
1. Acesse cloudinary.com e crie conta gratuita
2. No Dashboard, copie Cloud Name e API Key
3. Vá em Settings → Upload → Upload Presets → Add upload preset
4. Dê o nome sophia_store, defina Signing Mode: Unsigned e salve
5. Cole os dados acima e clique em Conectar
